my portfolio

Check out some of my most interesting projects!

Music Visualiser with Java

This project creates 5 different audio visuals using the programming languange Java, Java processing and Papplet. The audio visuals are responsive to the music being played, the different sounds like the level bass etc.

Django - Web Framework

This project uses Django to redesign a bad quality existing website and to also apply localisation and globalisation to it. Django is a free and open-source, Python-based web framework that follows the model–template–views architectural pattern.

BFS & DFS Algorithms

This project is a simple Java program that shows how a searching algorithm known as Breadth First Search and Depth First Search is implemented. BFS uses Queue data structure for finding the shortest path, while DFS uses Stack data structure.

Pygame Space Invader

This project is a simple retro style game called "Space Invader". I created this game with the Python programming language and the pygame module built into Python which is designed for writing video games. It is a game in which a spaceship has to destroy other spaceships in order to progress.

Live Diary - MSD Android Studio Project

Developed in Java with Android Studio, My Live Diary app is a streamlined app for note-taking and diary entries, saved with titles, dates, and times for easy organization. Styled as digital sticky notes, it offers a visually appealing and engaging way to capture thoughts, activities, or memories, combining convenience with style.

MedAssist - AI Medical Assistant

MedAssist, a health advice chatbot, leverages OpenAI's API to offer accessible health information. Users can query via text or speech-to-text, receiving advice and having responses read out loud. It generates downloadable medical reports and guides users to nearby healthcare facilities using real-time geolocation. Developed with MudBlazor, MedAssist combines AI with user-friendly design to improve healthcare accessibility.
